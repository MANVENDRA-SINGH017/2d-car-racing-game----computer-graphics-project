
This project presents the development of a simple 2D car racing game using the Python Turtle graphics library. The game simulates a moving road environment where the player controls a car to avoid collisions with incoming enemy vehicles. The objective is to survive as long as possible while achieving a high score.

The project demonstrates fundamental concepts of computer graphics such as animation, object movement, collision detection, and user interaction. The use of Turtle graphics makes it simple to visualize and implement real-time motion and event-driven programming.
